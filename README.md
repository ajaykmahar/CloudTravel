# CloudTravel Solution!

Follow the below steps to clone and run unit test cases.

## Clone git repo.

- Clone solution to your workspace.
	> **git clone https://github.com/ajaykmahar/CloudTravel.git**

-  **Install**.
	> **$pip install -r requirements.txt**

-  **Run test cases**.
	> $pytest
-  **Run test covrage **.
	> $pytest --cov=Solution
